NAME

Wombat::Globals - global constants

SYNOPSIS

DESCRIPTION

This package contains constants that are global to Wombat.

CONSTANTS

DEBUG
A flag determining whether messages of level 'DEBUG' are logged or not.

FORM_TYPE_URLENCODED
The content type used for HTTP form data

SERVER_INFO
The server name and version

SESSION_COOKIE_NAME
The name of the cookie used to transmit the session id between the client and the server

SESSION_PARAMETER_NAME
The name of the parameter used to transmit the session id between the client and the server
